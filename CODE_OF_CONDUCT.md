# Code of Conduct

AeonProof adopts the **Contributor Covenant, version 2.1**, as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

We pledge to make participation in the AeonProof community a respectful,
welcoming, and harassment-free experience for everyone. We focus on what is
best for the community, show empathy, and accept constructive feedback with
grace.

## Scope

This Code applies within all project spaces (repository, issues, discussions,
pull requests) and when an individual is representing the project in public.

## Enforcement

Concerns may be reported privately to the maintainers at
**conduct@aeonscript.org** *(mailbox setup in progress — interim:
mmdrame2017+aeonproof-conduct@gmail.com)*.

All reports will be reviewed and investigated promptly and fairly. Maintainers
are obligated to respect the privacy and security of any reporter.
